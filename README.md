# 📊 Superstore Sales Analysis

## 📌 Project Overview

This project analyzes a Superstore Sales dataset using Python, Pandas, and NumPy to extract valuable business insights. It includes data cleaning, preprocessing, feature engineering, KPI analysis, customer segmentation, and sales trend analysis to support data-driven decision-making.

The project demonstrates practical data analysis techniques commonly used in real-world business environments.

---

## 🚀 Features

* Data Cleaning
* Duplicate Removal
* Missing Value Handling
* Date Conversion
* Feature Engineering
* Total Sales Calculation
* Product-wise Sales Analysis
* Region-wise Sales Analysis
* City-wise Sales Analysis
* Business KPI Calculation
* Customer Segmentation
* Monthly & Yearly Sales Trend Analysis
* Export Processed Data to CSV and Excel

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* OpenPyXL
